HTML Activities Designer - Embedding Examples

This page demonstrates different ways to embed activities from HTML Activities Designer into your website.

Note: These examples assume you have created public activities in your HTML Activities Designer dashboard. Replace "sample-quiz" with your actual activity slug.

Method 1: JavaScript SDK (Recommended)

The easiest way to embed activities. The SDK handles everything automatically.

<!-- Include the SDK --> <script src="https://your-domain.railway.app/sdk/activities.js"></script> <!-- Create a container for your activity --> <div id="my-activity"></div> <!-- Render the activity --> <script> HTMLActivities.render('sample-quiz', 'my-activity'); </script>

Method 2: Auto-initialization

Even easier! Just add data attributes and the SDK will automatically initialize.

<!-- Include the SDK --> <script src="https://your-domain.railway.app/sdk/activities.js"></script> <!-- Activity will auto-load on page load --> <div data-html-activity="sample-quiz"></div>

Method 3: iframe Embedding

Safe for external sites and provides complete isolation.

<!-- Include the SDK --> <script src="https://your-domain.railway.app/sdk/activities.js"></script> <!-- Container --> <div id="iframe-activity"></div> <!-- Render as iframe --> <script> HTMLActivities.renderIframe('sample-quiz', 'iframe-activity', { width: '100%', height: '600px' }); </script>

Or use data attributes:

<div data-html-activity="sample-quiz" data-mode="iframe" data-width="100%" data-height="600px"></div>

Method 4: Direct iframe (No JavaScript)

If you can't use JavaScript, you can embed activities directly with an iframe.

<iframe src="https://your-domain.railway.app/api/embed/sample-quiz/render" width="100%" height="600px" frameborder="0" style="border-radius: 8px;"> </iframe>

Configuration Options

You can configure the SDK for different environments:

<script> // Configure for a different API base URL HTMLActivities.config({ apiBase: 'https://your-custom-domain.com' }); // Then render activities as normal HTMLActivities.render('sample-quiz', 'my-container'); </script>

Custom Styling

Activities come with default styling, but you can override it:

<style> .html-activity-container { /* Override default styles */ font-family: 'Your Custom Font', sans-serif; max-width: 600px; } .html-activity-container .question { /* Customize question appearance */ background: #your-brand-color; border-radius: 15px; } .html-activity-container .btn { /* Customize buttons */ background: #your-button-color; } </style>
